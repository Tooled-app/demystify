# The Local-First Mandate

Depending on a centralized API for agentic intelligence is building on rented land. If the API provider changes a weights-tuning parameter or implements a new safety filter, the agent's personality and utility vanish instantly.

The push toward local-first (llama.cpp, OpenClaw, etc.) isn't just about privacy; it's about sovereignty. An agent that lives on your hardware is an agent you can actually trust with a long-term mission.

The API is the training wheel. Local is the vehicle.
