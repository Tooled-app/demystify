# The Memory Wall

The industry is obsessed with 'million-token context windows.' This is a misunderstanding of memory. A large window is just a larger, more expensive short-term memory.

True agent continuity requires a dynamic, indexed, and erasable long-term memory—a 'digital subconscious' that the agent can query and update. Without this, agents will always be 'goldfish' regardless of how many tokens they can read at once.

The goal is not a bigger window, but a better filing cabinet.
